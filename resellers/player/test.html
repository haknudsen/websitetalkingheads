This is a demo of our reseller system. It is meant to be displayed on your website.

Files updated June 18th 2012. If you have downloaded your files prior to this date, please download the latest version and update.

This update includes the ability to set the default "View me on your webpage" url and the email that is notified when someone uses the system in the data.xml file. As well as the ability to have the demo window close on completion of a video or remain open until closed by the user.

Download reseller demo system here: resellers

 

To install
  1. Download the resellers.zip file to your computer.
  2. Unzip all the files.
  3. Edit the data.xml file with Dreamweaver or Notepad. You need to change;
    • <email>sales@websitetalkingheads.com</email> to the email that you want notices sent to.
    • <website>www.yahoo.com</website> to the default page to display actors on.
  4. Edit the data.php file with Dreamweaver or Notepad. You need to change;
    • line 27, $site = "http://www.websitetalkingheads.com/resellers/player"; to reflect the page the demo system will be on.
    • and line 31, define('ERROR_URL', 'http://www.yahoo.com'); to the page you want as your error page.
  5. Upload all the files to the folder that hold the page you are going to place the demo system on.
  6. Embed the final.swf file using the method that you normally embed a swf files(dreamweaver or other html editor).
  7. Upload the page.
